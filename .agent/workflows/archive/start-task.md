---
description: Initialize a new development task. Usage: /start-task [light] <task-name>
---

1.  **Parse Arguments**:
    - Use the first argument as `TASK_NAME`.

2.  **Initialize Artifacts**:
    - Create `task.md` in the root or `.agent/tasks/` if preferred.
    - Update `docs/TODO.md` to mark the item as in-progress.

3.  **Notify User**:
    - Confirm you are ready to start working on `<TASK_NAME>`.
